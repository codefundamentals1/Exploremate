## Run Locally

Clone the project

```bash
  git clone https://github.com/codefundamentals1/Exploremate_Ai_Itenenraryplanner.git
```

Go to the project directory

```bash
  cd Exploremate_Ai_Itenenraryplanner
```

Install dependencies

```bash
  npm install
```

Start the server

open two terminal 
Terminal 1 


```bash
  cd Backend
  npm start
```

Terminal 2
```bash
  cd Frontend
  npm run start
```
